• Transform: Change the shape.
    This is a CSS style property name.
  • Translate: It is a value of the transform style rule.
    It means you move either to X axis or Y axis or both axis.
  • Transition: How long it will take to do the change of the shape or color.
    There are four transition property:>>
    1. Property name -- width or height or margin etc etc
    2. Duration -- in seconds
    3. Easing function -- ease in or out
    4. Delay -- in seconds
